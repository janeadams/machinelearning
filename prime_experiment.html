Machine, Learning

An experiment by Jane Adams (🐦)

Computers don't actually understand math, they just compute solutions. But can machine learning enable computers to begin to grasp the theory of math?

We build a data set that comprises of all primes under 2,000 multiplied by one another to generate a list of subprimes (RSA numbers). We also generate a random selection of composite numbers, the same size as the semiprime set. We merge both into a single data set, with a binary classifier for semiprime or composite class. Using 20% training data and 80% testing data, we ask a decision tree whether a given set of numbers {a,b,c} is a multiplication of primes, where a*b=c. The model returns 98.3% accuracy.
