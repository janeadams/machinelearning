Machine, Learning

Computers can compute an equation, but they don't understand the math itself. Can machine learning teach computers to understand math?

We build a data set that comprises of all primes under 2,000 multiplied by one another, and a random selection of non-prime multiples matching the prime data set size. Using 20% training data and 80% testing data, we ask a decision tree whether a given set of numbers {a,b,c} is a multiplication of primes, where a*b=c. The model returns 98.3% accuracy.
